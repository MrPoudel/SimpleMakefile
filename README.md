# SimpleMakefile
This project demonstrates how to write a simple game in C program and build the project using a Makefile.

#Usuage

##Requirements

1. The GNU toolchain

2. GNU make

##Instructions

###Building

1. Clone the repository: https://github.com/MrPoudel/SimpleMakefile.git 

2. cd SimpleMakefile && make

3. ./quiz
